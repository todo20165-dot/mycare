# دليل الاختبار - MyCare

دليل شامل لاختبار جميع ميزات تطبيق MyCare.

## 🧪 أنواع الاختبارات

### 1. اختبارات الوحدة (Unit Tests)

```bash
# تشغيل جميع اختبارات الوحدة
php artisan test --filter=Unit

# اختبار نموذج معين
php artisan test tests/Unit/Models/UserTest.php

# اختبار متحكم معين
php artisan test tests/Unit/Controllers/AuthControllerTest.php
```

### 2. اختبارات التكامل (Integration Tests)

```bash
# تشغيل اختبارات التكامل
php artisan test --filter=Integration

# اختبار ميزة معينة
php artisan test tests/Integration/MedicationTest.php
```

### 3. اختبارات الميزات (Feature Tests)

```bash
# تشغيل اختبارات الميزات
php artisan test --filter=Feature

# اختبار ميزة محددة
php artisan test tests/Feature/AuthenticationTest.php
```

## 📋 قائمة الاختبارات اليدوية

### اختبار المصادقة

#### 1. التسجيل الجديد
- [ ] افتح صفحة التسجيل
- [ ] أدخل بيانات صحيحة
- [ ] اختر نوع الحساب
- [ ] اضغط "تسجيل"
- [ ] تحقق من إنشاء الحساب بنجاح
- [ ] تحقق من إعادة التوجيه إلى لوحة التحكم

#### 2. تسجيل الدخول
- [ ] افتح صفحة تسجيل الدخول
- [ ] أدخل بريد إلكتروني وكلمة مرور صحيحة
- [ ] اضغط "دخول"
- [ ] تحقق من الدخول بنجاح
- [ ] تحقق من ظهور بيانات المستخدم

#### 3. تسجيل الخروج
- [ ] من لوحة التحكم، اضغط على الملف الشخصي
- [ ] اضغط "تسجيل الخروج"
- [ ] تحقق من الخروج بنجاح
- [ ] تحقق من إعادة التوجيه إلى الصفحة الرئيسية

#### 3. حذف الدواء

#### 4. تسجيل الجرعات

## 🐛 الأخطاء الشائعة والحلول

### مشكلة: Service Worker لا يعمل
**الحل:**
1. تأكد من أن التطبيق يعمل على HTTPS أو localhost
2. امسح ذاكرة التخزين المؤقت
3. أعد تحميل الصفحة

### مشكلة: البيانات لا تُحفظ محلياً
**الحل:**
1. تحقق من تفعيل localStorage
2. تحقق من حجم البيانات
3. امسح ذاكرة التخزين المؤقت

### مشكلة: الإشعارات لا تظهر
**الحل:**
1. تحقق من تفعيل الإشعارات في المتصفح
2. تحقق من إعدادات الإشعارات في النظام
3. تحقق من الأخطاء في console
