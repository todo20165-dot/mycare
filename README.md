# MyCare - تطبيق إدارة الرعاية الصحية المنزلية

تطبيق ويب متكامل (PWA) لإدارة الرعاية الصحية المنزلية مع دعم كامل للعمل بدون اتصال بالإنترنت وتصميم يشبه تطبيقات الجوال الاحترافية.

## 🎯 المميزات الرئيسية

### 📋 إدارة الأدوية
- إضافة وتعديل وحذف الأدوية
- تسجيل الجرعات مع حالة الالتزام
- حساب معدل الالتزام تلقائياً
- سجل كامل للجرعات المأخوذة والمفقودة
- تنبيهات ذكية لمواعيد الأدوية

### ❤️ تتبع العلامات الحيوية
- تسجيل 6 أنواع من العلامات الحيوية
- كشف القراءات غير الطبيعية
- رسوم بيانية تفاعلية
- تحليل الاتجاهات

### 👥 نظام الارتباط العائلي
- ربط المرضى بأفراد العائلة
- نظام موافقة على الطلبات
- عرض بيانات المريض للأسرة المصرح لها

### 👨‍⚕️ لوحة تحكم الطبيب
- عرض قائمة المرضى
- إدارة الوصفات الطبية
- تتبع العلامات الحيوية
- إضافة ملاحظات طبية

### 🔐 لوحة تحكم الإدارة
- إدارة المستخدمين والأدوار
- إحصائيات شاملة
- إدارة الأدوية والإشعارات

### 📊 نظام التقارير
- تقارير الأدوية والالتزام
- تقارير العلامات الحيوية
- تصدير البيانات بصيغة PDF

### 🔔 نظام الإشعارات
- إشعارات تذكير الأدوية
- إشعارات العلامات الحيوية غير الطبيعية
- إرسال البريد الإلكتروني

### 📱 PWA والعمل بدون اتصال
- تطبيق قابل للتثبيت على الأجهزة الجوالة
- العمل الكامل بدون اتصال بالإنترنت
- مزامنة تلقائية عند العودة للاتصال

## 🛠️ المتطلبات التقنية

- **PHP:** 8.0 أو أعلى
- **Laravel:** 9.0 أو أعلى
- **قاعدة البيانات:** SQLite أو MySQL
- **متصفح:** يدعم PWA و Service Worker

## 📦 التثبيت والإعداد

### 1. استنساخ المشروع
```bash
git clone <repository-url>
cd mycare-app-laravel
```

### 2. تثبيت المتطلبات
```bash
composer install
```

### 3. إعداد البيئة
```bash
cp .env.example .env
php artisan key:generate
```

### 4. إعداد قاعدة البيانات
```bash
# إنشاء ملف قاعدة البيانات SQLite
touch database/database.sqlite

# تشغيل الـ Migrations
php artisan migrate
```

### 5. تشغيل التطبيق
```bash
php artisan serve
```

ثم افتح المتصفح على: `http://localhost:8000`

## 🚀 الاستخدام الأساسي

### تسجيل حساب جديد
1. انقر على "إنشاء حساب جديد"
2. أدخل بيانات الحساب
3. اختر نوع الحساب
4. اضغط "تسجيل"

### إضافة دواء
1. من الصفحة الرئيسية، اضغط على "الأدوية"
2. اضغط على "+ إضافة دواء جديد"
3. أدخل بيانات الدواء
4. اضغط "حفظ"

### تسجيل الجرعات
1. من قائمة الأدوية، اختر الدواء
2. اضغط على "تناول" للجرعة المجدولة
3. سيتم تسجيل الجرعة تلقائياً

### تسجيل العلامات الحيوية
1. من الصفحة الرئيسية، اضغط على "العلامات الحيوية"
2. اضغط على "+ تسجيل قياس جديد"
3. اختر نوع القياس وأدخل القيم
4. اضغط "حفظ"

## 📱 تثبيت التطبيق كـ PWA

### على الهاتف الذكي
1. افتح التطبيق في المتصفح
2. اضغط على قائمة المتصفح (⋮)
3. اختر "تثبيت التطبيق" أو "إضافة إلى الشاشة الرئيسية"
4. اضغط "تثبيت"

### على سطح المكتب
1. افتح التطبيق في Chrome أو Edge
2. انقر على أيقونة التثبيت في شريط العنوان
3. اضغط "تثبيت"

## 🌐 العمل بدون اتصال

التطبيق يدعم العمل الكامل بدون اتصال:
- عرض الأدوية والعلامات الحيوية المحفوظة
- تسجيل الجرعات الجديدة
- تسجيل القياسات الجديدة
- قراءة الإشعارات

عند العودة للاتصال، سيتم مزامنة جميع البيانات تلقائياً.
